# MowFlow
### Elevate Grounds Company — Daily Cash Flow Tracker

A mobile-first business tracker for daily revenue, labor costs, Ohio taxes, workers comp, and overhead allocation.

## Install on Android
1. Open the app URL in Chrome
2. Tap the three-dot menu → "Add to Home screen"
3. Tap Add — it installs as a full-screen app

## Install on iPhone
1. Open the app URL in Safari
2. Tap the Share button → "Add to Home Screen"
3. Tap Add
